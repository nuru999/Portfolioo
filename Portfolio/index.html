My Journey in Technology

Building secure and meaningful digital systems

My name is Nuru Amudi, a software developer with a strong interest in cybersecurity. My journey into technology started with learning computer fundamentals through ICDL, which gave me a strong foundation in digital skills.

I later studied software engineering where I developed practical experience in web development using HTML, CSS, and JavaScript while learning how modern web systems are built.
